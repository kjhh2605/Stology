## 사용한 AI/제작 도구

| 도구 | 활용 내용 |
|---|---|
| Claude | 기획안 인터뷰, 요구사항 정리, 서비스 방향성 구체화 |
| OpenAI Codex | 프로젝트 문서 정리, 데모사이트 구현 보조, 작업 지침(`AGENTS.md`) 관리 |
| oh-my-codex(OMX) | 데모사이트 제작, 작업 흐름 오케스트레이션, 검증 관리 |
| HyperFrames | Stology 60초 소개 영상 구성, HTML 기반 모션/렌더링 제작 |
| Figma MCP | PM 와이어프레임 제작, 화면설계서 기준 화면 구조 정리 |
| Notion MCP | 기획 자료 정리, 산출물 관리, 프로젝트 문서화 보조 |
| AI 기반 문서/기획 보조 | MVP 범위 정리, 화면 흐름 점검, 용어와 산출물 일관성 검토 |
