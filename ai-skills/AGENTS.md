# Stology AI 스킬 보관본 작업 지침

이 파일은 `ai-skills/` 아래 프로젝트 진행 중 만든 AI 스킬 보관본에 적용됩니다. 루트 `AGENTS.md`의 공통 프로젝트 관리 지침을 함께 따릅니다.

## 역할

`ai-skills/`는 프로젝트 중 작성하거나 조정한 스킬을 보관하는 로컬 아카이브입니다. 활성 Codex 스킬 설치 경로가 아니라, 재사용 가능한 지침 원본을 보존하는 공간으로 취급합니다.

## 편집 원칙

- 스킬을 수정할 때는 대상 `SKILL.md`가 단독으로 읽혀도 목적, 트리거, 입력, 출력, 검증 기준이 이해되게 유지합니다.
- 프로젝트 전용 맥락과 범용 스킬 지침을 섞을 때는 범위를 명확히 구분합니다.
- 기존 스킬의 트리거 문구, 파일 구조, 상대 경로를 바꿀 때는 해당 스킬을 사용하는 문서나 작업 흐름에 영향이 없는지 확인합니다.
- 절대 경로, 개인 환경 경로, 임시 파일 경로는 필요할 때만 넣고, 가능하면 상대 경로나 플레이스홀더를 사용합니다.
- 스킬 보관본을 수정해도 설치된 스킬이 자동으로 바뀌지는 않는다는 점을 완료 보고에 명확히 남깁니다.

## 검증

스킬 수정 후 최소한 다음을 확인합니다.

- `SKILL.md`의 목적과 트리거가 서로 맞는지 확인
- 상대 경로가 실제 파일 구조와 맞는지 확인
- 프로젝트 전용 지침이 범용 스킬로 오해되지 않게 표현되었는지 확인
