# 01. 로그인 화면

> 와이어프레임 제작용 디자인 및 기능 요구사항

---

## 화면 개요

| 항목 | 내용 |
|---|---|
| 화면명 | 로그인 화면 |
| 화면 유형 | 풀스크린 페이지 |
| URL | `/login` |
| 권한 | 비로그인 (인증 없음) |
| 진입 경로 | 미인증 상태에서 모든 페이지 접근 시 / 로그아웃 직후 |

---

## 레이아웃 구조

```
┌──────────────────────────────────────────┐
│                                          │
│                                          │
│              [서비스 로고]                 │
│              "Stology"                   │
│                                          │
│       "당신의 활동이                       │
│        소중한 자산이 되도록"                │
│                                          │
│                                          │
│         ┌──────────────────┐             │
│         │ 카카오 로그인 버튼   │             │
│         └──────────────────┘             │
│                                          │
│                                          │
└──────────────────────────────────────────┘
```

- 중앙 정렬, 세로 배치
- 좌측 사이드바, 우측 상단 프로필 없음 (로그인 전)
- 단일 CTA로 시선 집중

---

## 화면 구성 요소

### 1. 서비스 로고
- **위치**: 상단 중앙
- **내용**: Stology 로고 + 텍스트
- **클릭 동작**: 없음

### 2. 슬로건
- **위치**: 로고 아래
- **내용**: "당신의 활동이 소중한 자산이 되도록"
- **스타일**: 부제 텍스트, 한 줄

### 3. 카카오 로그인 버튼
- **위치**: 슬로건 아래 중앙
- **내용**: 카카오 디자인 가이드 준수 (카카오 노란색 #FEE500)
- **레이블**: "카카오로 시작하기" 또는 "카카오 로그인"
- **클릭 동작**: 카카오 OAuth 외부 페이지로 이동

---

## 기능 요구사항

### 인증 흐름

```
[로그인 화면]
   ↓ 카카오 로그인 버튼 클릭
[카카오 OAuth 외부 페이지]
   ↓ 인증 완료
[홈 화면]
```

### 인증 후 라우팅

| 사용자 구분 | 도착 화면 |
|---|---|
| 신규 사용자 (첫 로그인) | 홈 화면 |
| 기존 사용자 | 홈 화면 |
| 초대 링크로 진입 중인 사용자 | 해당 스터디 페이지 (자동 참여) |

신규/기존 사용자 구분 없이 동일하게 홈으로 진입한다.

### 인증 정책

- 카카오 소셜 로그인 단일 방식
- 모든 페이지 로그인 필수 (비로그인 시 이 화면으로 리다이렉트)
- 이메일/비밀번호, 다른 소셜 로그인 미지원

---

## 인터랙션 명세

| 인터랙션 | 동작 |
|---|---|
| 카카오 로그인 버튼 클릭 | 카카오 OAuth 외부 페이지 이동 |
| 카카오 인증 성공 | 홈 화면 또는 초대받은 스터디 페이지로 라우팅 |
| 카카오 인증 실패 | MVP 미정의 (카카오 기본 에러 처리에 위임) |

| 상태 | 처리 |
|---|---|
| 카카오 인증 실패 | MVP 미정의 |
| 네트워크 끊김 | MVP 미정의 |

---

## 디자인 가이드

### 시각 우선순위
1. 로고 + 슬로건 (상단)
2. 카카오 로그인 버튼 (메인 CTA)

### 톤
- 미니멀
- 단일 CTA에 시선 집중
- 추가 요소 (회원가입 링크, 약관 등) 없음

### 반응형
- 웹 전용 (모바일 미고려)
