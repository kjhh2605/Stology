## 스터디 페이지 공통 헤더

모든 스터디 페이지 탭이 공유하는 헤더 구조:

```
┌─────────────────────────────────────────────────────────────┐
│ [스터디명]   [현재 N주차]  [멤버 프로필들]  [시작일]   [⚙]   │
├─────────────────────────────────────────────────────────────┤
│ [지식 구조][자료 업로드][주차별 기록][주차별 리포트][질문함]    │
└─────────────────────────────────────────────────────────────┘
```

> 헤더, 탭 네비게이션, ⚙ 설정 드롭다운 및 스터디 종료 흐름 등의 자세한 명세는 `07-study-container.md` 참조

## 지식 구조 탭 레이아웃 구조

```
┌─────────────────────────────────────────────────────────────┐
│ [스터디 페이지 공통 헤더]                                       │
├─────────────────────────────────────────────────────────────┤
│ [필터: 전체 ▼] [주차별 필터: 전체 ▼]                            │
├─────────────────────────────┬───────────────────────────────┤
│                             │                               │
│                             │   [우측 사이드 패널]            │
│                             │   (노드 클릭 시 슬라이드 인)     │
│      [3D 그래프 캔버스]       │                               │
│                             │   - 노드명                     │
│                             │   - 정의                       │
│                             │   - 활성화 자료 목록            │
│                             │   - 관계별 연결 노드            │
│                             │                               │
│                             │                               │
└─────────────────────────────┴───────────────────────────────┘
```

### 영역 비율
- 그래프 캔버스: 메인 영역
- 사이드 패널: 노드 클릭 시 우측에서 슬라이드 인 (약 30~35% 너비)
- 사이드 패널이 닫혔을 때는 그래프 전체 영역 사용

## 화면 구성 요소

### 1. 상단 필터 영역

#### 뷰 필터
- **A. 전체 활성 노드 보기** (기본)
- **B. 특정 주차 활성 노드 보기** (드롭다운에서 주차 선택)
  - 선택 주차에 활성화된 노드만 강조
  - 나머지는 회색 반투명

### 2. 3D 그래프 캔버스 (메인 영역)

#### 노드 시각 표현

| 상태 | 시각 표현 |
|---|---|
| 비활성 | 회색 반투명 |
| 활성 | 단일 색상 (전체 노드 동일색) |
| 활성 강도 | 자료 수에 비례한 색상 진하기 |
| 강조 | 선택 노드와 연결된 노드들 (테두리 강조 등) |

> 활성화 강도는 단순 on/off가 아닌 단계적 표현. 자료가 많이 연결된 노드일수록 색상이 진해짐.

#### 엣지 (관계선)
- 단순 선 (방향성 없음)
- 별도 클릭 액션 없음

#### 그래프 조작
| 조작 | 동작 |
|---|---|
| 드래그 (빈 공간) | 그래프 회전 |
| 스크롤 | 줌 인/아웃 |
| 노드 드래그 | 일시 이동, 놓으면 원위치 복귀 |

### 3. 노드 인터랙션

#### 활성 노드 클릭
- 우측 사이드 패널 슬라이드 인
- 해당 노드와 연결된 다른 노드들 강조 (시각적 하이라이트)

#### 비활성 노드 클릭
- 클릭 가능
- 사이드 패널에 노드 정의만 표시 (자료 목록 비어 있음)

#### 빈 공간 클릭
- 사이드 패널 슬라이드 아웃
- 강조 해제

### 4. 우측 사이드 패널

#### 활성 노드 선택 시
```
┌────────────────────────────┐
│ [X]                        │
│                            │
│  노드명                     │
│  ──────────────────        │
│                            │
│  노드 정의/설명              │
│  (운영진 사전 정의 텍스트)    │
│                            │
│  ──────────────────        │
│                            │
│  관련 자료 (3개)            │
│  ┌──────────────────────┐  │
│  │ 자료 1 미리보기 ...   │  │
│  │ 자료 2 미리보기 ...   │  │
│  │ 자료 3 미리보기 ...   │  │
│  └──────────────────────┘  │
│  ↑ 섹션 클릭 시 팝업 호출    │
│                            │
│  ──────────────────        │
│                            │
│  관계별 연결 노드            │
│  선행: A, B, C, ...         │
│  관련: D, E, F, ...         │
│  [더보기]                   │
│                            │
└────────────────────────────┘
```

#### 비활성 노드 선택 시
```
┌────────────────────────────┐
│ [X]                        │
│                            │
│  노드명                     │
│  ──────────────────        │
│                            │
│  노드 정의/설명              │
│                            │
│  ──────────────────        │
│                            │
│  관련 자료 (0개)            │
│  (아직 활성화되지 않음)      │
│                            │
│  ──────────────────        │
│                            │
│  관계별 연결 노드            │
│  선행: A, B, C, ...         │
│  관련: D, E, F, ...         │
│  [더보기]                   │
│                            │
└────────────────────────────┘
```

#### 사이드 패널 표시 정보

| 항목 | 활성 노드 | 비활성 노드 |
|---|---|---|
| 노드명 | ⭕ | ⭕ |
| 노드 정의/설명 | ⭕ | ⭕ |
| 활성화시킨 자료 목록 (시간순) | ⭕ | 비어 있음 |
| 관계별 연결 노드 리스트 | ⭕ | ⭕ |

#### 자료 미리보기 섹션
- 시간순 정렬 (최신 또는 오래된 순 — MVP 미정의, 기본 최신순 추정)
- 미리보기 형태 (제목 또는 첫 줄 등 일부 표시)
- **섹션 클릭 시 → 원본 자료 조회 팝업** (파생 화면 참조)

#### 관계별 연결 노드 리스트
- 운영진 온톨로지 관리 화면에서 정의된 관계 종류에 따라 그룹핑
- 노드명만 쉼표 구분으로 나열
- 화면 너비 기준 줄바꿈
- "더보기" 버튼 클릭 시 전체 노출

## 기능 요구사항

### 그래프 초기 상태
- 스터디 생성 직후: 선택한 온톨로지 템플릿의 전체 노드가 **회색 반투명**으로 표시
- 노드 위치는 자동 레이아웃 (force-directed 등)

### 활성화 상태 갱신
- 자료 업로드 → AI 추출 → 검토 확정 시 자동 반영
- 활성화 강도는 검토 확정된 자료 수에 비례

### 종료된 스터디 처리
- 그래프 인터랙션 가능 (조회만)
- 사이드 패널의 자료 클릭 가능 (다운로드 가능)
- 새 자료 추가 불가 (다른 탭에서 차단)

| 인터랙션 | 동작 |
|---|---|
| 활성 노드 클릭 | 사이드 패널 슬라이드 인 + 연결 노드 강조 |
| 비활성 노드 클릭 | 사이드 패널 슬라이드 인 (자료 영역 비어 있음) |
| 빈 공간 클릭 | 사이드 패널 슬라이드 아웃 + 강조 해제 |
| 노드 드래그 | 일시 이동 → 놓으면 원위치 |
| 그래프 빈 공간 드래그 | 회전 |
| 스크롤 | 줌 |
| 자료 미리보기 섹션 클릭 | 원본 자료 조회 팝업 |
| 사이드 패널 X 버튼 클릭 | 슬라이드 아웃 |
| 주차 필터 드롭다운 선택 | 해당 주차 활성 노드만 강조 |
| 관계 연결 노드 "더보기" | 전체 노드명 노출 |

## 디자인 가이드

### 시각 우선순위
1. 3D 그래프 (메인 시각 요소)
2. 사이드 패널 (선택 시)
3. 상단 필터 (보조)

### 색상 정책
- **활성 노드**: 단일 색상 (서비스 메인 컬러 또는 강조 색)
- **비활성 노드**: 회색 반투명 (낮은 채도)
- **강조 노드**: 테두리 강조 또는 글로우
- **관계선**: 흐린 회색 (노드보다 약하게)

### 사이드 패널 디자인
- 우측에서 슬라이드 인/아웃 애니메이션
- 그래프와 명확히 분리되는 배경

### 종료된 스터디
- 헤더에 "종료됨" 배지 추가
- 그래프 색상은 유지 (조회 가능 의미)

## 표시 정보

### 팝업 헤더
- "{노드명} 관련 자료" 또는 "노드명 — 자료 목록"
- X 닫기 버튼

### 자료 항목별 표시 정보
| 항목 | 내용 |
|---|---|
| 파일명 | 자료 제목 |
| 업로드자 | 작성자 이름 (프로필 사진 옵션) |
| 업로드일 | YYYY-MM-DD |
| 다운로드 버튼 | 파일 다운로드 액션 |
| 링크 버튼 | 마크다운 뷰어 새 탭 (MVP 보류) |

### 정렬
- 시간순 (사이드 패널과 동일)

### 스크롤
- 팀원 자료 업로드 리스트가 많을 경우 세로 스크롤
