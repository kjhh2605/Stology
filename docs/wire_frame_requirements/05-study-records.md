## 화면 구성 요소

### 1. 주차 토글 영역

- 토글 형식 (탭 또는 칩 형태)
- 한 번에 하나의 주차만 선택 가능
- 1주차 ~ 현재 주차 노출
- 미래 주차는 노출 안 됨 (또는 비활성화)
- 선택된 주차 시각 강조

### 2. 노드 리스트 영역

#### 노드 항목 표시
- 노드명
- 펼침/접힘 화살표 (▶ / ▼)
- 인라인 펼침 가능

#### 노드 표시 범위
- 선택한 주차에 **새로 활성화된 노드 + 기존 노드 중 보강된 노드**

#### 정렬
- 가나다 순 / 알파벳 순

---

## 노드 인라인 펼침 영역

노드 클릭 시 그 노드 아래에 자료 리스트가 인라인으로 펼침.

### 자료 항목 표시 정보

| 항목 | 내용 |
|---|---|
| 파일명 | 자료 제목 |
| 업로드자 | 작성자 이름 |
| 업로드일 | YYYY-MM-DD |
| 다운로드 버튼 | 파일 다운로드 |
| 링크 버튼 | 마크다운 뷰어 새 탭 (MVP 보류) |

### 인라인 펼침 동작

- 노드 클릭 → 자료 리스트가 인라인 펼침
- 다시 클릭 → 접힘
- 여러 노드 동시 펼침 가능 (각자 독립적)

### 자료 정렬
- 시간순 (MVP 미명시, 기본 최신순 추정)

---

## 기능 요구사항

### 주차 선택
- 토글 클릭 시 해당 주차로 즉시 전환
- URL 쿼리 파라미터 등으로 상태 유지 가능 (MVP 미명시)

### 노드 표시 로직
- "활성화" = 그 주차에 처음 활성화된 노드
- "보강" = 기존 활성 노드에 그 주차에 추가 자료가 매핑된 경우
- 둘 모두 같이 표시

### 자료 액션
- 다운로드: 파일 다운로드
- 링크: 마크다운 뷰어 새 탭 (MVP 보류)

## 인터랙션 명세

| 인터랙션 | 동작 |
|---|---|
| 주차 토글 클릭 | 해당 주차 선택, 노드 리스트 갱신 |
| 노드 클릭 (접힘 상태) | 인라인 펼침, 자료 리스트 노출 |
| 노드 클릭 (펼침 상태) | 접힘 |
| 자료 항목 다운로드 버튼 | 파일 다운로드 |
| 자료 항목 링크 버튼 | 마크다운 뷰어 (MVP 보류) |

## 빈 상태

| 케이스 | 처리 |
|---|---|
| 선택 주차 활성 노드 0개 | "이 주차에 활성화된 노드가 없습니다" 메시지 |

---

## 종료된 스터디 처리

- 조회만 가능
- 모든 인터랙션 (주차 선택, 노드 펼침, 다운로드) 동일하게 작동
- 추가 자료 업로드는 자료 업로드 탭에서 차단됨

---

## 디자인 가이드

### 주차 토글
- 칩(chip) 또는 탭 형태
- 가로 나열, 가로 스크롤 가능 (주차 많을 경우)
- 선택 상태: 배경색 강조 또는 밑줄

### 노드 항목
- 펼침 아이콘 (▶ / ▼)
- 호버 시 시각 변화
- 펼침 시 자료 리스트 영역 들여쓰기

### 자료 항목
- 카드 형태 또는 단일 행
- 다운로드/링크 버튼 명확히 구분
