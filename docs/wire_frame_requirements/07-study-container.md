# 07. 스터디 페이지 컨테이너 (헤더 + 탭 + 스터디 관리)

> 와이어프레임 제작용 디자인 및 기능 요구사항
> 파생 화면: ⚙ 설정 드롭다운, 검토 인원 수 조정 모달, 스터디 종료 확인 모달, 스터디 종료 안내 화면, 초대 링크 생성 모달(⚙ 진입 케이스)

---

## 화면 개요

| 항목 | 내용 |
|---|---|
| 화면명 | 스터디 페이지 컨테이너 |
| 화면 유형 | 페이지 (5개 탭 컨테이너) |
| URL | `/study/{id}` |
| 권한 | 스터디 멤버 (스터디장 + 스터디원) |
| 진입 경로 | 홈 카드 클릭 / 사이드바 스터디 클릭 / 초대 링크 진입 |

본 문서는 모든 스터디 탭의 공통 헤더와 스터디 관리(설정/종료) 흐름을 다룬다. 각 탭의 본문 명세는 별도 문서(03~06, 08) 참조.

---

## 페이지 구조

```
┌─────────────────────────────────────────────────────────────┐
│  [사이드바]              │ [우측 상단 프로필]                 │
│                          │                                   │
│                          │ ┌─────────────────────────────┐   │
│                          │ │ [헤더: 스터디 메타 + ⚙]      │   │
│                          │ │                              │   │
│                          │ │ [탭 네비게이션]              │   │
│                          │ ├─────────────────────────────┤   │
│                          │ │                              │   │
│                          │ │   [현재 탭 본문 영역]         │   │
│                          │ │                              │   │
│                          │ │                              │   │
│                          │ └─────────────────────────────┘   │
└─────────────────────────────────────────────────────────────┘
```

---

## 스터디 페이지 공통 헤더

### 레이아웃

```
┌─────────────────────────────────────────────────────────────┐
│ [스터디명]   [현재 N주차]  [멤버 프로필들]  [시작일]   [⚙]   │
├─────────────────────────────────────────────────────────────┤
│ [지식 구조][자료 업로드][주차별 기록][주차별 리포트][질문함]    │
└─────────────────────────────────────────────────────────────┘
```

### 헤더 표시 정보

| 요소 | 내용 |
|---|---|
| 스터디명 | 스터디 생성 시 입력된 이름 (최대 20자) |
| 현재 주차 | 시작일 기준 자동 계산 ("3주차" 형태) |
| 멤버 프로필 | 참여 중인 스터디 멤버의 프로필 사진들 (가로 배치) |
| 시작일 | YYYY-MM-DD 형태 |
| ⚙ 설정 아이콘 | 스터디장만 노출, 종료된 스터디에서는 미노출 |

### 멤버 프로필 표시

- 프로필 사진을 가로로 나열
- 일정 인원 초과 시 "+N" 형태로 표시 (예: 5명까지 노출, 그 이상은 "+3")
- 호버 시 이름 툴팁 (선택 사항)
- 멤버 프로필 클릭 동작: MVP 미정의

### 종료된 스터디 헤더 차이

- 스터디명 옆에 "종료됨" 배지 표시
- ⚙ 설정 아이콘 미노출
- 시작일 옆에 종료일 추가 표시

### 탭 구성 (왼쪽부터)

| 1 | 지식 구조 | `03-study-knowledge.md` |

| 3 | 주차별 기록 | `05-study-records.md` |
| 4 | 주차별 리포트 | `06-study-report.md` |

### 기본 진입 탭

- 스터디 페이지 진입 시 **지식 구조 탭**이 기본 활성
- URL: `/study/{id}` → 지식 구조 탭
- 직접 URL: `/study/{id}/upload`, `/study/{id}/records` 등

### 탭 전환 동작

- 클릭한 탭의 본문 영역만 갱신
- 헤더는 그대로 유지
- 페이지 새로고침 없음

# 파생 화면 1. ⚙ 설정 드롭다운

## 개요

| 항목 | 내용 |
|---|---|
| 화면명 | ⚙ 설정 드롭다운 |
| 화면 유형 | 드롭다운 메뉴 |
| 진입 경로 | 스터디 페이지 헤더 ⚙ 아이콘 클릭 |
| 권한 | 스터디장만 (다른 멤버에게는 ⚙ 아이콘 자체가 미노출) |

## 레이아웃

```
                                          [⚙] ← 클릭
                                         ┌──────────────┐
                                         │ 초대 링크    │
                                         │ 검토 인원 수  │
                                         │ 스터디 종료  │
                                         └──────────────┘
```

## 항목 (자주 쓰는 순)

| # | 항목 | 클릭 시 동작 |
|---|---|---|
| 1 | 초대 링크 | 초대 링크 생성 모달 호출 (파생 화면 2) |
| 2 | 검토 인원 수 조정 | 검토 인원 수 조정 모달 호출 (파생 화면 3) |
| 3 | 스터디 종료 | 스터디 종료 확인 모달 호출 (파생 화면 4) |

## 닫기 동작

- 드롭다운 외부 클릭 → 자동 닫힘
- 항목 선택 → 닫히면서 해당 액션 수행

## 종료된 스터디 처리

- ⚙ 아이콘 자체가 미노출 → 본 드롭다운에 접근할 수 없음

# 파생 화면 2. 초대 링크 생성 모달 (⚙ 진입)

## 개요

| 항목 | 내용 |
|---|---|
| 화면명 | 초대 링크 생성 모달 |
| 화면 유형 | 모달 (오버레이) |
| 진입 경로 | ⚙ 설정 드롭다운 → "초대 링크" 클릭 |
| 권한 | 스터디장만 |

> 참고: 스터디 생성 직후 진입 케이스는 `02-home.md`에 정의됨. 본 문서는 스터디 페이지에서 진입한 경우만 다룸.

## 레이아웃

```
┌─────────────────────────────────────┐
│  스터디 초대              [X]         │
├─────────────────────────────────────┤
│                                     │
│  ┌─────────────────────────────┐    │
│  │ https://stology.com/inv/... │    │
│  └─────────────────────────────┘    │
│                                     │
│              [복사하기]              │
│                                     │
│              [닫기]                  │
└─────────────────────────────────────┘
```

## 구성

- 모달 제목: "스터디 초대"
- 초대 링크 URL (읽기 전용 텍스트 박스, 전체 선택 가능)
- 복사 버튼
- 닫기 버튼 (또는 X)

## 액션

| 버튼 | 동작 |
|---|---|
| 복사하기 | 클립보드에 URL 복사 + 토스트 알림 ("링크가 복사되었습니다") |
| 닫기 / X | 모달 닫힘 → 스터디 페이지 복귀 |

## MVP 제약

- 링크 재발급 불가 (1회 발급, 영구 유지)
- 만료 기간 없음

# 파생 화면 3. 검토 인원 수 조정 모달

## 개요

| 항목 | 내용 |
|---|---|
| 화면명 | 검토 인원 수 조정 모달 |
| 화면 유형 | 모달 (오버레이) |
| 진입 경로 | ⚙ 설정 드롭다운 → "검토 인원 수 조정" 클릭 |
| 권한 | 스터디장만 |

## 레이아웃

```
┌─────────────────────────────────────┐
│  검토 인원 수 조정          [X]       │
├─────────────────────────────────────┤
│                                     │
│  검토 인원 수                        │
│                                     │
│        ┌──┐    ┌──┐    ┌──┐         │
│        │ -│    │ 2│    │ +│         │
│        └──┘    └──┘    └──┘         │
│                                     │
│  변경된 인원 수는 새로 시작되는       │
│  검토에만 적용됩니다.                 │
│                                     │
│              [닫기]                  │
└─────────────────────────────────────┘
```

## 구성

- 모달 제목: "검토 인원 수 조정"
- 현재 검토 인원 수 표시 (중앙 숫자)
- 증감 버튼 (- / +)
- 안내 메시지: "변경된 인원 수는 새로 시작되는 검토에만 적용됩니다"
- 닫기 버튼

## 동작

| 버튼 | 동작 |
|---|---|
| **+** 버튼 | 인원 수 +1, 즉시 적용 + 토스트 알림 |
| **-** 버튼 | 인원 수 -1, 즉시 적용 + 토스트 알림 |
| 닫기 / X | 모달 닫힘 |

## 검증 규칙

| 케이스 | 처리 |
|---|---|
| 최소값 | 1명 (그 이하로 감소 시 - 버튼 비활성) |
| 최대값 | 현재 스터디 멤버 수 (그 이상 증가 시 + 버튼 비활성) |
| 즉시 적용 | 저장 버튼 없이 클릭 즉시 변경 반영 |

## 변경 영향 정책

- 변경 시점부터 **새로 시작되는 검토에만 적용**
- **진행 중인 검토는 기존 인원 수 유지**
- 예: 현재 2명 → 3명 변경 시, 이미 시작된 검토는 2명 기준으로 진행

# 파생 화면 4. 스터디 종료 확인 모달

## 개요

|---|---|
| 화면명 | 스터디 종료 확인 모달 |
| 화면 유형 | 모달 (오버레이) |
| 진입 경로 | ⚙ 설정 드롭다운 → "스터디 종료" 클릭 |
| 권한 | 스터디장만 |

## 레이아웃

```
┌─────────────────────────────────────┐
│  스터디 종료              [X]         │
├─────────────────────────────────────┤
│                                     │
│  스터디를 종료하시겠습니까?           │
│                                     │
│  스터디를 종료하면 더 이상            │
│  자료 업로드, 검토, 질문 작성이      │
│  불가능합니다.                       │
│                                     │
│         [아니오]  [예]               │
└─────────────────────────────────────┘
```

## 구성

- 모달 제목: "스터디 종료"
- 메인 메시지: "스터디를 종료하시겠습니까?"
- 안내 메시지: "스터디를 종료하면 더 이상 자료 업로드, 검토, 질문 작성이 불가능합니다"
- 액션 버튼: "예" / "아니오"

## 액션

| 버튼 | 동작 |
|---|---|
| 예 | 스터디 종료 처리 → 스터디 종료 안내 화면(파생 화면 5)으로 이동 |
| 아니오 | 모달 닫힘, 변경 없음 |
| X | 모달 닫힘, 변경 없음 |

## 위험 액션 처리

- 종료는 되돌릴 수 없음 (재개 불가)
- 텍스트 입력 확인 등 추가 절차는 없음 (단순 Yes/No)

# 파생 화면 5. 스터디 종료 안내 화면

## 개요

| 항목 | 내용 |
|---|---|
| 화면명 | 스터디 종료 안내 화면 |
| 화면 유형 | 풀스크린 페이지 |
| URL | `/study/{id}/ended` (또는 별도 라우트, 1회성) |
| 진입 경로 | 스터디 종료 확인 모달 → "예" 클릭 직후 자동 진입 |
| 권한 | 스터디장 (해당 스터디의) |
| 재진입 가능 | 불가 (1회성) |

## 레이아웃

```
┌────────────────────────────────────────────┐
│                                            │
│           [축하/마무리 일러스트]            │
│                                            │
│         "스터디가 종료되었습니다"           │
│                                            │
│       이번 스터디에서 함께한 활동:          │
│                                            │
│       총 활성 노드:        N개             │
│       업로드된 자료:       N개             │
│       작성된 질문:         N개             │
│                                            │
│                                            │
│            [홈으로 이동]                    │
│                                            │
└────────────────────────────────────────────┘
```

## 구성

- 메인 메시지: "스터디가 종료되었습니다"
- 종료 요약 통계:
  - 총 활성 노드 수
  - 업로드된 자료 수
  - 작성된 질문 수
- CTA 버튼: "홈으로 이동"
- 일러스트 또는 마무리 시각 요소 (선택)

## 액션

| 버튼 | 동작 |
|---|---|
| 홈으로 이동 | 홈 화면(`/`)으로 라우팅 |

## 재진입 규칙

- 본 화면은 **종료 직후 1회만** 표시
- 닫고 나면 다시 접근 불가
- 이후 사이드바 "지난 활동" → 종료된 스터디 클릭 시 **곧장 종료된 스터디 페이지(조회 모드)로 진입**

---

## 스터디 종료 전체 흐름

```
[스터디 페이지] → ⚙ 클릭
   ↓
[⚙ 설정 드롭다운] → "스터디 종료" 클릭
   ↓
[스터디 종료 확인 모달] → "예" 클릭
   ↓
[스터디 종료 안내 화면] → "홈으로 이동" 클릭
   ↓
[홈 화면]
   - 사이드바: 종료된 스터디는 "지난 활동" 섹션으로 이동
   - 진입점 카드 영역: 해당 스터디 카드 제거

(이후) 사이드바 "지난 활동" → 종료된 스터디 클릭
   ↓
[종료된 스터디 페이지 - 조회 모드 직행]
```

## 종료된 스터디 페이지 조회 모드 정책

종료된 스터디 페이지 진입 시 다음 정책이 적용된다:

### 가능한 액션
- 자료 다운로드
- 지식 구조 그래프 조회
- 주차별 기록 조회
- 주차별 리포트 조회
- 질문/답글 조회
- 자료 원본 조회 팝업

### 불가능한 액션 (UI에서 미노출 또는 비활성)
- 자료 업로드 (자료 업로드 탭의 업로드 영역 미노출)
- 자료 검토 (검토 필요/수정하기 상태 클릭 시 반응 없음)
- 질문 작성 (질문함의 "질문 작성" 버튼 미노출)
- 답글 작성 (답글 작성 입력창 미노출)
- 질문/답글 수정/삭제 (본인 작성건도 불가)
- ⚙ 설정 아이콘 미노출
